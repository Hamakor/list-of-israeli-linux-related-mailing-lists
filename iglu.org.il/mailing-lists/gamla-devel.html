Gamla Development

Description

This mailing-list is dedicated to the development of Gamla, the RAD and WWW application server for perl.

Posting Guidelines

Whatever is related to the development and usage of Gamla: discussions, suggestions, questions, answers, ideas, RFCs, and so on.

Subscribing to the Mailing-List

Send an empty mail message to the following address:

gamla-devel-subscribe@yahoogroups.com

Unsubscribing from the Mailing-List

Send an empty mail message to the following address:

gamla-devel-unsubscribe@yahoogroups.com

Posting Messages to the Mailing-List

Send your messages to the following address:

gamla-devel@yahoogroups.com

Contacting the Mailing-List's Owner

Send messages to the mailing-list owner to the following address:

gamla-devel-owner@yahoogroups.com
