Israeli Web Standards Mailing List

Description

Mailing list for increasing the standards' compatibility of Israeli web-sites and making sure they operate correctly in Linux and cross-platform browsers.

Posting Guidelines

Whatever related to supprting web standards in Israeli sites.

Subscribing to the Mailing-List

Send an empty mail message to the following address:

w3c-subscribe@projects.hamakor.org.il

Unsubscribing from the Mailing-List

Send an empty mail message to the following address:

w3c-unsubscribe@projects.hamakor.org.il

Posting Messages to the Mailing-List

Send your messages to the following address:

w3c@projects.hamakor.org.il

Contacting the Mailing-List's Owner
