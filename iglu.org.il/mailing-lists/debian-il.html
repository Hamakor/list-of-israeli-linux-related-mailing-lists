Debian-IL

Description

A group dedicated for users of the Debian Linux distribution in Israel.

Posting Guidelines

Coordination of Debian activities in Israel. Local Debian support and advocacy. Questions of localization of Debian to Israel

Subscribing to the Mailing-List

Send an empty mail message to the following address:

debian-il-subscribe@debian.org.il

Unsubscribing from the Mailing-List

Send an empty mail message to the following address:

debian-il-unsubscribe@debian.org.il

Posting Messages to the Mailing-List

Send your messages to the following address:

debian-il@debian.org.il

Contacting the Mailing-List's Owner

Send messages to the mailing-list owner to the following address:

debian-il-owner@debian.org.il
