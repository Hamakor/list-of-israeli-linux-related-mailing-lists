Linux-IL

Description

The main mailing-list for Linux users in Israel.

Posting Guidelines

Linux-related questions and discussions. No newbie questions (use gnubies-il instead).

Post size is limited to 16 kilobytes. Larger posts are queued for moderator approval.

If you are not subscribed the message that you send to the list will require moderator approval and will not be sent to the members automatically.

Subscribing to the Mailing-List

Send a message containing the following line:

subscribe

To the following address:

linux-il-request@linux.org.il

Unsubscribing from the Mailing-List

Send a message containing the following line:

unsubscribe

To the following address:

linux-il-request@linux.org.il

Posting Messages to the Mailing-List

Send your messages to the following address:

linux-il@linux.org.il

Contacting the Mailing-List's Owner

Send messages to the mailing-list owner to the following address:

linux-il-owner@linux.org.il
