Linux-IL Announce

Description

Announcements for Linux users in Israel (Linux-IL members are automatically subscribed)

Posting Guidelines

This list is moderated and the moderator accepts any Linux-related announcements. Note that one is already subscribed to it if he is subscribed to Linux-IL

Subscribing to the Mailing-List

Send a message containing the following line:

subscribe

To the following address:

linux-il-announce-request@linux.org.il

Unsubscribing from the Mailing-List

Send a message containing the following line:

unsubscribe

To the following address:

linux-il-announce-request@linux.org.il

Posting Messages to the Mailing-List

Send your messages to the following address:

linux-il-announce@linux.org.il

Contacting the Mailing-List's Owner

Send messages to the mailing-list owner to the following address:

linux-il-announce-owner@linux.org.il
