Translation List

Description

Mailing List dedicated to translation of technical or semi-technical documents from English to Hebrew or Arabic.

Posting Guidelines

Whatever related to translation of documents. Posts partially in Hebrew are acceptable.

Subscribing to the Mailing-List

To subscribe write a message with the following line as body:

subscribe translation

to the following address:

majordomo@ivrix.org.il

Unsubscribing from the Mailing-List

To unsubscribe write a message with the following line as body:

unsubscribe translation

to the following address:

majordomo@ivrix.org.il

Posting Messages to the Mailing-List

Send your messages to the following address:

translation@ivrix.org.il

Contacting the Mailing-List's Owner

Send messages to the mailing-list owner to the following address:

nyh@math.technion.ac.il
